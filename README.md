## 项目概述

视频去水印工具是一个专为Windows用户设计的桌面应用程序，采用现代化的图形界面，能够帮助用户快速去除抖音、快手、小红书等平台视频的水印。本工具从v2.0.0版本开始，从命令行界面升级为图形用户界面，提供更加直观友好的操作体验。

### 核心优势

- 🎨 **现代化GUI**：采用PyQt6开发的图形界面，扁平玻璃化设计风格
- 🖥️ **无边框设计**：透明背景和自定义样式，视觉效果更佳
- 🎯 **一键操作**：简单几步即可完成视频去水印
- 📱 **多平台支持**：全面支持主流短视频平台
- 📥 **本地下载**：支持直接下载无水印视频到本地
- 🎭 **图标显示**：完美解决图标显示问题
- 🖼️ **封面预览**：实时显示视频封面图片
- ⚙️ **自定义设置**：支持下载路径等个性化配置

## 使用方法

### 基本使用步骤（图形界面）

1. **启动程序**：双击exe文件启动图形界面
2. **选择平台**：在界面中选择视频平台（抖音/快手/小红书）
3. **粘贴链接**：将分享文本或视频链接粘贴到输入框
4. **处理视频**：点击"去水印"按钮开始处理
5. **查看结果**：在结果区域查看视频信息和封面
6. **保存视频**：点击"下载内容"按钮保存到本地

### 界面操作说明

#### 平台选择
- 在主界面顶部的平台选择区域
- 点击对应的平台按钮（抖音/快手/小红书）
- 界面会根据选择自动调整
- 显示当前选择平台的示例链接

#### 链接输入
- 在中央的输入框中粘贴视频链接或分享文本
- 程序会自动识别其中的视频链接
- 支持多种格式的分享文本
- 智能提取纯链接

#### 处理操作
- 点击"去水印"按钮开始处理
- 界面会显示处理进度
- 完成后显示视频标题、封面等信息

#### 结果查看
- 结果区域显示视频标题
- 实时加载并显示封面图片
- 提供多个操作按钮：
  - **下载内容**：下载无水印视频
  - **打开封面**：在新窗口中预览封面图片
  - **复制链接**：复制无水印视频链接

#### 设置功能
- 点击设置按钮打开设置对话框
- 可以自定义下载路径
- 点击浏览按钮选择文件夹
- 设置自动保存

## 已知问题

### 当前版本已知限制

1. **网络依赖**：需要稳定的网络连接访问API
2. **平台限制**：仅支持抖音、快手、小红书，其他平台暂不支持
3. **大文件处理**：超大视频文件处理时间较长
4. **系统兼容**：当前exe文件仅支持Windows系统
5. **封面加载**：部分视频封面可能加载较慢

### 解决方案

1. **网络问题**：检查网络连接，或等待网络恢复后重试
2. **平台扩展**：后续版本将支持更多平台
3. **大文件处理**：建议在网络良好环境下下载大文件
4. **跨平台支持**：可考虑使用源码方式在其他系统运行
5. **封面问题**：耐心等待封面加载，或直接下载视频
