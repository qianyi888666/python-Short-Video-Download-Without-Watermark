## 项目概述

视频去水印工具是一个专为Windows用户设计的桌面应用程序，能够帮助用户快速去除抖音、快手、小红书等平台视频的水印。本工具采用Python开发，通过PyInstaller打包成独立的可执行文件，无需安装Python环境即可直接运行。

### 核心优势

- 🎯 **一键操作**：简单几步即可完成视频去水印
- 📱 **多平台支持**：全面支持主流短视频平台
- 🚀 **交互式界面**：直观友好的用户界面
- 📥 **本地下载**：支持直接下载无水印视频到本地
- 🔄 **智能切换**：多API地址自动切换，确保服务稳定
- 🎭 **模拟模式**：无需网络连接即可体验完整功能
